# Task Manager Application - Frontend

This is the frontend portion of the Full-Stack Task Manager Application. It is built using React, Vite, and Tailwind CSS. 

For full documentation covering both the frontend and backend, please see the [Main Project README](../README.md) located in the root directory.

## Available Scripts

In the project directory, you can run:

### `npm run dev`

Runs the app in the development mode.\
Open [http://localhost:5173](http://localhost:5173) to view it in your browser.

The page will reload when you make changes.\
You may also see any lint errors in the console.

### `npm run build`

Builds the app for production to the `dist` folder.\
It correctly bundles React in production mode and optimizes the build for the best performance.
